# 首次登入帳號

首次登入必須綁定帳號資料和強制更新密碼。

## 綁定帳號流程

1. 輸入個人 e-mail 及強制更新密碼。
   ![首次登入](img/bind-account.png)

2. 點擊送出後，要去檢查個人信箱
   ![認證 e-mail](img/vertify-email.png)

3. 點擊 e-mail 內連結
   ![e-mail信件](img/vertify-email-content.png)

4. 認證成功，帳號才能開始使用。
   ![e-mail 認證成功](img/vertify-email-seccessed.png)
