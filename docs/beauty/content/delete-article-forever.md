# 永久刪除文章

:::danger 謹慎操作
此操作不可還原，請小心操作！！<br/>
:::

1.  從文章列表點擊回收桶，進入文章回收桶頁面
    ![文章列表](img/go-to-article-trash.png)
    ![文章回收桶](img/article-trash.png)

2.  選擇要永久刪除的文章
    ![選擇文章](img/select-softdelete-article.png)

3.  永久刪除文章
    ![永久刪除文章](img/delete-article-forever.png)
