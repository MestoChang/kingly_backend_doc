# 如何新增关键字

1. 进入关键字列表

![关键字列表](img/keyword-list-01.png)

2. 点击右上，新增关键字

![关键字列表](img/keyword-list-02.png)

3. 出现填写关键字栏位

![新增关键字栏位](img/add-keyword.png)

4. 点击 `送出`，系统检查没有重复关键字即新增成功

![新增关键字成功](img/add-keyword-success.png)

:::warning
如果同时新增多个关键字，只要有一个关键字重复即无法送出！
:::
