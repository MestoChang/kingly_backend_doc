# 身分可以刪除嗎

<!-- TODO 補截圖 -->

## 限制

已綁定帳號的身份無法刪除。

## 操作流程

1. 進入身份權限管理頁面
   ![刪除身分](img/delete-a-role-01.png)

2. 選擇要刪除的身份，點擊 刪除
   ![刪除身分](img/delete-a-role-02.png)

3. 若此身份已有綁定帳號，會顯示無法刪除
   ![刪除身分](img/delete-a-role-03-02.png)

4. 無綁定帳號的身份即可成功刪除
   ![刪除身分](img/delete-a-role-03-01.png)
