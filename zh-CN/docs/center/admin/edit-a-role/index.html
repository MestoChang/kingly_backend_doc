編輯身分的功能權限

  1. 進入身份權限管理，選擇要編輯的身份

  2. 進入設定身份頁面，欄位的部分僅有名稱可修改

  3. 可調整身份的功能權限 因為這個身份是最高的 A 級帳號，所以基本上什麼權限都可設定。

    參考 管理員帳號及身分權限說明，編輯權限有帳號等級的區分與限制。

  4. 保存變更
